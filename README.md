# AutomationTestStore
A hands-on automation project simulating real-world e-commerce flows using Selenium and Pytest. Built to deepen my understanding of UI testing, locator strategies, and test architecture.

## Project Overview
This project is to test the UI flow of the demo website automationteststore.com for educational purposes.


## Installation and execution
1. Setup virtual environment: `python -m venv venv`
2. Activate virtual environment:
```
venv/Scripts/activate
 ```
3. Install dependencies: `pip install selenium`

### Tools used:
- Python
- Selenium Webdriver
- Pytest-HTML
- Faker 
- Page Object Model(POM)
