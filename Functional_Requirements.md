# Functional Requirements of Automation Test Store
---
<!-- FR categories -->

## User Account management:

**Fr-001**: User is able to register an account <br>
**Fr-002**: User is able to login to account <br>
**Fr-003**: User is able to logout

---
## Product search and discovery:

**Fr-004**: User is able to search, filter, and browse items

---

## Cart Operations:

**Fr-005**: User is able to add and remove items from cart <br>
**Fr-006**: User is able to select the desired amount of the item

---

## Stock Validation:

**Fr-007**: User is able to detect when an item is out of stock <br>
**Fr-008**: User is not able to select stock that exceeds the maximum amount per customer. <br>
**Fr-009**: User is not able to select stock that is below the minium amount per customer.

---
## Currency Conversion:

**Fr-010**: User is able to switch between avilable currencies <br>
**Fr-011**: User is able to see the price of the item updated after currency conversion.

---
## Checkout Process:

**Fr-012**: User is able to review items in cart <br>
**Fr-013**: User is able to enter their address for item delivery <br>
**Fr-014**: User is able to checkout
